# console_game
java -jar RunCry.jar

Game figures:

Player is black 'o' symbol.

Walls is pimk '#' symbol.

Enemies is red 'X' symbol.

Escape gate is blue 'O' symbol. 

Player should escape from enemies to escape gate. 

How to move: 

  'w' - up. 
  
  's' - down. 
  
  'a' - left. 
  
  'd' - right.
  
Player should confirm the movement with Enter key.
